Reposition ThumbGate around its core mission: **stop paying for the same AI mistake twice.** Frontier-model tokens are expensive and repeat hallucinations, retry loops, and known-bad tool calls compound the bill. ThumbGate's Pre-Action Gates intercept the call before it reaches the model so zero tokens get spent on mistakes you've already corrected once.

- New `scripts/token-savings.js` helper computes estimated tokens + dollars saved using a configurable model mix (Sonnet-blended by default).
- New live "💸 Tokens Saved" counter on the dashboard that turns blocked-gate counts into a real-time savings estimate.
- Rewritten README hero, landing page hero, OG/meta tags, package description, plugin description, and GitHub repo "About" copy to lead with the token-cost angle.
- New blog post `docs/marketing/blog-token-cost-mission.md` and pre-validated social pack `docs/marketing/token-cost-mission-social-pack.md` (X/Threads/LinkedIn/HN/Reddit/video).
