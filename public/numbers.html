Methodology

Where the numbers come from. This page is regenerated from local scripts — no survey data, no hand-edited figures, no third-party attribution. Every number on this page is produced by code in the public ThumbGate repo.

  • Active checks — union of shipped default rules and the auto-promotion ledger (auto).
  • Actions blocked/warned — sum of occurrences across gates with the corresponding action.
  • Hours saved — conservative 15-minute/incident estimate for debugging a repeated AI mistake × (blocks + warns).
  • Dollars saved — blended per-call token estimate (2k input + 600 output) × blocks × 2026-04-15 Anthropic + OpenAI list prices. See scripts/token-savings.js for the full price snapshot.
  • Bayes error rate — irreducible classifier error of the current risk scorer given its feature set. High values mean "add features, don't tune thresholds."

Last auto-promotion: none. Regenerated on every release via npm run numbers:generate and on a weekly cadence.
