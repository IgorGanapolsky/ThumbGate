FAQ

Is ThumbGate only for Amp?

No. Amp is one supported workflow, but the same feedback and enforcement loop also works across Claude Code, Cursor, Codex, Gemini, and OpenCode.

Why use guardrails instead of harness instructions alone?

Harness instructions are advisory. ThumbGate pre-action checks intercept the tool call itself and block the known-bad pattern before execution.
