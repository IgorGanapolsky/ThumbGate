What OpenAI shipped

The updated Agents SDK moved up the stack: model-native harnesses, configurable memory, native sandbox execution, manifest-based workspaces, and durable snapshotting for long-running tasks.

That matters because more teams now have a viable execution layer for agents without stitching together a custom harness from scratch.

FAQ

Does ThumbGate replace OpenAI Agents SDK?

No. OpenAI Agents SDK is the harness and execution foundation. ThumbGate is the enforcement layer that turns operator feedback into searchable memory, prevention rules, and pre-action gates.

Why add ThumbGate if OpenAI already added sandbox execution?

Sandbox execution contains code after it runs. ThumbGate decides whether the risky action should run in the first place, then carries the lesson forward so the same mistake is less likely to be attempted again.
