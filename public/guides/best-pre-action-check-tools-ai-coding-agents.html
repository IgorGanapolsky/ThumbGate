FAQ

Are pre-action checks different from code review?

Yes. Code review happens after work exists. A pre-action check runs before the AI coding agent executes the risky action.

Can ThumbGate block bad tool calls in AI agents?

Yes. ThumbGate is designed to block known-bad tool calls, require approval for risky actions, and log lower-risk events for audit.

Does this replace CI?

No. CI proves the result. ThumbGate governs the action before the result exists. Use both.
