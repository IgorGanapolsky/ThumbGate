# Contributing

## Development

```bash
npm test
npm run start:api
npm run start:mcp
```

## Pull Request Requirements

1. Tests must pass.
2. API and adapter contracts must remain aligned.
3. No secrets in commits.
4. Update docs if behavior changes.

## Budget Rule

This project targets a hard `$10/month` external runtime budget.
Changes that increase spend must include explicit budget impact notes.
